# 🎨 Catálogo de Tintas  

Aplicação web mobile-first desenvolvida para facilitar a visualização de cores de tintas disponíveis em loja, permitindo que clientes tenham uma noção mais realista do produto aplicado na parede antes da compra.

O projeto foi pensado para uso principalmente em dispositivos móveis, considerando o contexto real de atendimento via WhatsApp em uma loja de materiais de construção, onde clientes tinham dificuldade em entender as cores disponíveis apenas por fotos do catálogo físico.

## 💡 Problema  

No atendimento remoto, clientes frequentemente:  

- Não conseguiam visualizar corretamente as cores do catálogo físico  
- Tinham dificuldade em imaginar a tinta aplicada na parede  
- Ficavam inseguros para decidir a compra à distância  

Isso tornava o processo de venda menos assertivo.

---

## ✅ Solução  

Criação de um catálogo digital com visualização de tintas em contexto de aplicação, permitindo que o cliente veja a cor de forma mais próxima da realidade.

Ao longo do desenvolvimento, novas funcionalidades foram incorporadas para melhorar a experiência de navegação e busca no catálogo, como:

- Barra de busca  
- Carrossel de produtos  
- Organização por categorias
